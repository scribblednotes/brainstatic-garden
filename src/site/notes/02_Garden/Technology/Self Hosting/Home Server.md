# 🌱 Home Server

I've only recently gotten into self hosting, so I'm not very experienced with it at all.  However, with some guidance from the many online sources and a little help from AI, I've managed to turn an old Dell Vostro 3590 laptop into my very own home server.  I've even gotten things to work on it too!  

Self hosting is very gratifying, if time consuming and sometimes infuriating.  It's reassuring to know that I have a private place to store my data in these times of massive corporate interference in our lives.  I hope to expand on this "hobby" a lot more in the future.
